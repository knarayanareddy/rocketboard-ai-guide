========================================================================
GLOSSARY OF KEY TERMS
========================================================================
- Pack: The primary boundary of isolation and curation. A collection of diverse knowledge sources representing an engineering domain.
- Source: An upstream integration entity (e.g., a Confluence space, a GitHub repo, a Slack channel) connected to a Pack.
- Chunk: The smallest atomic unit of knowledge ingested and embedded (stored in `knowledge_chunks` table).
- Span: A reference to a slice of a chunk or multiple chunks used during retrieval to hydrate evidence.
- Generation: A monotonically increasing ID marking a complete re-index or ingestion job. Used for pin/swap visibility.
- Grounding Gate: The strictly enforced SLO that runs post-LLM generation to verify that technical claims are backed by cited Spans and strips hallucinations.
- Detective Loop: An iterative multi-hop retrieval process (agentic loop) that rewrites queries based on intermediate evidence.
- Symbol Graph (Graph v2): A semantic layer mapping definitions and references for deterministic code navigation (stored in `symbol_definitions`, `symbol_references`).
- Rollup: Aggregated daily/weekly analytical data (stored in `pack_quality_daily`) used for the Trust Dashboard.
